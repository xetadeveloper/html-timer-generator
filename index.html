Email Timer Generator

How to use

  • The Date start type asks if you want to use the current date or you want to provide a specific start date
  • The Time run type asks if you want to run the timer to a set date or if you want to run the timer for some specified time (days, hours, minutes, seconds).

When you choose your settings, click on the Generate code button to generate the timer code, which you can copy to your clipboard and use anywhere.

NOTE: This generates full HTML code (HTML + CSS + JS), so any changes needed to be made to the template can be made in the styles section of the HTML, but leave the script untouched for the functionality to work properly
